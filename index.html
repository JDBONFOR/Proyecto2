Este HTML es para probar conexión a GITHub

Pasos que ejecutamos:

  1. pushd para posicionarme en la carpeta en la Terminal
  2. git init
  3. git remote add origin "ruta dada por GIT para el repositorio"
  4. git add .
  5. git commit -m “mensaje a agregar de versionado, preferible en inglés y profesional."
  6. git push -u origin master
